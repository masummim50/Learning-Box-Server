Learning box api

Endpoints

Login

use only google sign in method from firebase, returns some user info and an accesstoken, set that to authorization header, will need it for every request

GET

to get personal post, make sure to send the accesstoken in the authorization header

To get all the posts that are marked public

for all the get requests, you can send page, search and tags query.
page by default is 1, and limit is by default 10. returned data will have a totalCount property

POST

send the post object in req.body, returns the new post

PATCH

make sure to send both the post id and the user id for the update operation, returns the updated document

DELETE

send post id to delete a specific post
